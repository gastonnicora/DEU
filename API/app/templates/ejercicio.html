Ejercicios
Campos de la clase ejercicio:
id integer not null //Generado automáticamente
nombre string not null
descripcion string
tiempo integer
tipo_tiempo string //Tipo de tiempop, por ejemplo min, seg
repeticiones integer //Cantidad de repeticiones de un ejercicio
tipo integer not null //Si es por tiempo (0) o por repeticiones(1)
borrado integer not null //borrado logico, por default= 0, 1 es que esta borrado
categoria integer not null //Categoria de ejercicio: calentamiento(0), entrenamiento(1), regenerativo(2)
video integer not null //id del video
publico integer not null //si la aplicación esta publica, por default= 0, 1 es que esta en privado
